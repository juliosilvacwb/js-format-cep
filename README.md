# js-format-cep

Lib to format CEP

#### Javascript example:
```js
const cep = require('js-format-cep');  
console.log(cpf.formatCEP("88829120"))
``` 
Output:
88.829-120

#### Typescript example:
```ts 
import { formatCEP } from 'js-format-cep'
console.log(formatCEP("88829120")
```

Output:
88.829-120
